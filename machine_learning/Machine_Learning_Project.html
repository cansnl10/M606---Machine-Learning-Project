House Price Prediction in Ankara¶

Author: Dogan Can Senol
Project: M606 - Machine Learning Final Project
Dataset: 100 samples of houses from 8 districts in Ankara
Goal: Predict house prices using machine learning

1) Problem Statement¶

A large real estate company located in Ankara, the capital of Türkiye, wants to develop a machine learning model that can automatically make predictions according to different districts and physical characteristics of the house in order to better analyze the market and offer more accurate prices to its customers. In Türkiye, it can be difficult to make price predictions, especially due to the free market in this area and the excessive pricing of some sellers. Therefore, this necessary project will be carried out to increase customer satisfaction and offer more rational offers to customers compared to competitors. Within the scope of this project, the information of 100 current apartments taken from 8 major districts in Ankara has been turned into a data set and the prediction of house prices has been formulated as a machine learning problem. Since the model will produce numerical output, not categorical, we can say that it is a regression problem.

10) Conclusion¶

In this project, it was aimed to predict house prices using a dataset consisting of 100 samples from 8 different districts in Ankara. Machine learning processes such as data preprocessing, exploratory data analysis, feature engineering, modeling and model evaluation were completed. Three different regression models were used, namely "Linear Regression", "Random Forest Regressor", "K-Nearest Neighbors Regressor". As a result of the comparisons made, the linear regression model showed the best performance in all metrics. This model reached the lowest MAE and RMSE values. It also reached the highest R-Square value and successfully explained 70%+ of price changes. It should be said that the feature engineering part did not benefit much. Although new variables and categorical groupings were added, these features did not improve the model performance. In general, it can be concluded that the Linear Regression model did not do a bad job on a dataset with little data.
